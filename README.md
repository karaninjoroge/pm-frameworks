# PM Frameworks Library

**Author:** Karani Njoroge
**Purpose:** A reference library of Product Management frameworks — when to use them, how to apply them, and common mistakes.

---

## Why This Library Exists

Frameworks are tools, not rules. A PM who applies RICE to every prioritization decision without thinking is as ineffective as one who never uses a framework at all. This library documents not just how each framework works, but when to reach for it and when to put it down.
